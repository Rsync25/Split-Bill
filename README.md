- ✅ Smart contract deployed on RSK Testnet
- ✅ Create bills with multiple participants
- ✅ Pay bills with RBTC
- ✅ Settle bills when all paid

## How to Test
1. Connect MetaMask to RSK Testnet
2. Get tRBTC from faucet
3. Create a bill with 2 participants
4. Pay from each account
5. Settle the bill
